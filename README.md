Retail Store Inventory & Chatbot App

A Flask web application for managing a retail store’s inventory with Supabase as the backend. Includes inventory management, low stock alerts, sales tracking, and a simple chatbot.

Features Inventory Management – view all products. Low Stock Alerts – see items with quantity < 5. Sales Management – record sales and update stock automatically. Chatbot – responds to greetings, inventory, stock, and sales queries, sales history.

Setup Clone the repository git clone cd

Create .env file SUPABASE_URL=your_supabase_url SUPABASE_KEY=your_supabase_anon_key
